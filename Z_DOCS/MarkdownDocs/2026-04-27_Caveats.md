Caveat Summary for Pre-2018-19 Financial Data
1. Source Data Availability
OSPI does not publish district-level per-pupil expenditure (PPE) data prior to 2019-20 in the same structure as the current PPE dataset.
Because of this, PPE values for 2014-15 through 2018-19 cannot be directly obtained from the published dataset.
2. Synthetic PPE Reconstruction (2014-15 ? 2018-19)
To maintain a continuous financial trend aligned with the performance data (which begins in 2014-15), earlier PPE values are synthetically reconstructed using a conservative, transparent method:
* 2018-19 PPE = 2019-20 PPE ū 0.80
(20% reduction to reflect pre-COVID, pre-ESSER cost levels)
* Each prior year = previous year ū 0.95
(5% annual reduction to reflect typical pre-COVID cost patterns)
This method produces a smooth, conservative downward trend consistent with historical expenditure behavior.
3. Synthetic Data Isolation
All reconstructed PPE values are stored in a separate table (SyntheticFinanceFact).
The original OSPI-published PPE data remains untouched in FinanceFact.
A unified view merges the two for analysis, ensuring:
* auditability
* reversibility
* clear separation of real vs. synthetic data
* easy replacement if historical F-196 data is integrated later
4. Limited Synthetic Fields
Synthetic rows include only the fields required for system-level analysis:
* Enrollment
* CostPerStudent (PPE)
* TotalExpenditures (Enrollment ū PPE)
No attempt is made to fabricate local, state, or federal expenditure components.
5. Transparency and Future Replacement
This synthetic baseline is a placeholder, not a claim of exact historical cost.
If OSPI releases pre-2019-20 PPE in a compatible format, or if F-196 data is later integrated, the synthetic rows can be replaced without altering the system's structure.

AI:  Data Suppression and Estimated Values
Washington's Office of Superintendent of Public Instruction (OSPI) suppresses certain assessment results to protect student privacy. Suppression occurs when reporting a value would reveal, or allow someone to infer, the performance of a very small number of students. This includes cases where the number of students meeting or not meeting standard is ten or fewer, or where other publicly reported values could be combined to derive that number.
When OSPI suppresses a result, the underlying count and percentage are not published. In some situations, however, other publicly available information provides enough context to form a reasonable estimate of the suppressed value. When this occurs, our system may display an estimated value only for the "All Students" group, and the estimate is clearly flagged. Estimated values are used solely to support trend visualization and cost modeling; they are not presented as official OSPI-reported results.
Our goal is to make long-term patterns easier to understand while remaining fully transparent about where data is reported, where it is suppressed, and where an estimate is used.
